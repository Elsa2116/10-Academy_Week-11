# Forecasting Financial Inclusion in Ethiopia

**Selam Analytics** | Financial Technology Consulting — Emerging Markets Division

A comprehensive forecasting system tracking Ethiopia's digital financial transformation using time series analysis, event impact modeling, and interactive visualization.

---

## Overview

Ethiopia is undergoing rapid digital financial transformation. Telebirr has grown to 54M registered users since launching in 2021, M-Pesa entered with 10M users, and for the first time P2P digital transfers have surpassed ATM cash withdrawals. Yet the 2024 Global Findex survey shows only 49% of Ethiopian adults have a financial account.

This system forecasts two core Global Findex dimensions of financial inclusion:
- **Access** — Account Ownership Rate
- **Usage** — Digital Payment Adoption Rate


Run in order: task1 → task2 → task3 → task4


- **Forecasts** — 2025–2027 projections with confidence intervals and scenarios

## Dataset Schema

The `ethiopia_fi_unified_data.csv` uses a **unified schema** where all records share the same columns:

| `record_type` | Count | Description |
|--------------|-------|-------------|
| `observation` | 30 | Measured values (Findex surveys, operator reports, infrastructure data) |
| `event` | 10 | Policies, product launches, market entries, milestones |
| `impact_link` | 14 | Modeled relationships between events and indicators |
| `target` | 3 | Official policy goals (NFIS-II) |

**Key design principle**: Events are NOT pre-assigned to pillars — their effects are captured through `impact_link` records, keeping the data unbiased.

## Key Findings

1. **2021–2024 slowdown is real but explained**: Telebirr's 54M registrations ≠ 9.45% Findex mobile money ownership — a ~10:1 ratio. Already-banked users adopted Telebirr without becoming new inclusion cases.

2. **Gender gap halving**: Female account ownership rose from 36% to 44% (2021→2024) while male ownership slightly fell; but a 14pp digital payment gender gap persists.

3. **P2P > ATM crossover**: For the first time in 2023, P2P digital transfers (ETB 180B) exceeded ATM withdrawals (ETB 160B) — a behavioral landmark.

4. **2025 NFIS-II target at risk**: Base scenario projects ~51–52% by 2025, missing the 55% target by ~3–4pp. Only optimistic scenario meets it.

5. **Digital payments growing faster**: Usage adoption is accelerating more than access, projecting 42–48% by 2027 — tracking toward the 2030 target of 50%.

## Methodology

- **Access model**: OLS linear regression on 5 Findex data points (2011–2024), R² > 0.95
- **Usage model**: OLS on reconstructed historical series (2014–2024)
- **Event effects**: Exponential saturation curves with documented lags; 40% realization rate (calibrated against 2021–2024 mobile money growth)
- **Scenarios**: Pessimistic (trend only −3.5pp), Base (trend + event uplift), Optimistic (full events + trend +3.5pp)
- **Evidence**: Comparable country studies: M-Pesa Kenya, Aadhaar India, Ghana/Tanzania telecom competition
